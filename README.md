# Spam Email Classifier

## Overview
Built an NLP-based classifier to detect spam messages using machine learning.
Trained on the UCI SMS Spam Collection dataset containing 5,500+ messages.

## Tech Stack
- Python, Pandas, Scikit-learn
- TF-IDF Vectorization
- Naive Bayes & Logistic Regression

## Results
- Naive Bayes Accuracy: 97.8%
- Logistic Regression Accuracy: 98.1%

## Key Steps
1. Data cleaning and preprocessing
2. TF-IDF feature extraction
3. Model training and comparison
4. Evaluation using confusion matrix and classification report

## Dataset
UCI SMS Spam Collection Dataset
